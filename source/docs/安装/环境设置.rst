环境设置
========================================

我们需要为Lumos进行环境配置，Lumos默认安装于用户目录下，请在~/.bashrc文件中添加如下内容

```shell
export PATH=/home/用户名/lumos/include/:$PATH
export PATH=/home/用户名/lumos/bin:$PATH
export LD_LIBRARY_PATH=/home/用户名/lumos/lib:$LD_LIBRARY_PATH
```

添加完成后使用如下命令激活

```shell
source ~/.bashrc
```

并使用

```shell
lumos --version
```

验证lumos框架是否安装成功
